# Configuração do Supabase

## 1. Configuração das Variáveis de Ambiente

Cadastre no arquivo `.env` suas credenciais do Supabase:

```env
SUPABASE_URL=sua_url_do_supabase
SUPABASE_KEY=sua_chave_do_supabase
```

## 2. Criação da Tabela de Produção

Execute o seguinte SQL no Supabase para criar a tabela `producao`:

```sql
create table public.producao (
  id bigint generated by default as identity not null,
  created_at timestamp with time zone not null default now(),
  "RSS" real null,
  "GB" real null,
  "RI" real null,
  "DataFoto" timestamp with time zone null default now(),
  "FotoFosso" character varying null,
  "Total" double precision null,
  constraint producao_pkey primary key (id)
) TABLESPACE pg_default;
```
